# Problem Definition

ADAS testing processes, especially in parking assistance systems, are often inefficient due to:

- Repeated setup of identical test configurations
- Lack of standardized configuration handling
- High resource consumption (e.g., engine runtime in stationary testing)
- Limited traceability of test conditions

These inefficiencies lead to increased testing time, higher costs, and unnecessary environmental impact.

The goal is to systematically analyze and improve these processes.
