# ADAS testing process Optimization


This repository contains the conceptual and analytical work for a bachelor thesis focused on process optimization in automotive ADAS testing environments.

## Focus

- Analysis of current testing workflows
- Identification of inefficiencies and resource waste
- Development of optimized process structures
- Evaluation of sustainability aspects

## Context

The work is based on practical experience in automotive testing, particularly in parking assistance systems.

## Goal

To develop a structured and reproducible approach for improving efficiency and sustainability in testing processes.

## Relation to Tooling

This repository is complemented by a separate implementation project:
-> ADAS Configuration Management

which provides a technical foundation for automation and configuration handling.
